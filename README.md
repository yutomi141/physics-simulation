# Physics Simulation

Numerical simulations for physics-informed informatics.
This repository serves as a record of my transition from generalist to specialist.

## Project Contents
- **control_theory/**
  - Servo PD Control: Step response analysis of a 2nd-order servo system with experimental parameter identification.
- **numerical_methods/**
  - Monte Carlo Integration: Numerical integration via random sampling.
  - Bayesian Estimation: Recursive Bayesian update for a constant unknown.
  - Kalman Filter: Dynamic state estimation with system/observation noise.
- **statistical_mechanics/**
  - Ising Model: Monte Carlo simulation for phase transitions.
  - Maxwell-Boltzmann Distribution: Velocity distribution of gas molecules.

## Environment
- Python 3.13.6
- NumPy / Matplotlib
